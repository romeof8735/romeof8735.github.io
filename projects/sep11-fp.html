Freedom Project

I used Kaboom.js to create a fun platformer game that has special powerups. I chose this because I wanted to create a game where every level would get more and more difficult as you progress. To help the player advance their would be power up abilities to help me contiune through.

For this Freedom Project we had to pick a tool, and using that tool we research about it and create something. In my case I picked Kaboom,js which allows for you to makes games. We also had to create a google slides presentation and basically sum up everything that we did and how we did it using code snippets here and their to fit in a google slides presentation without being too long.

My takeaways from this project is that it is always important to apply the correct functions and elements correctly so you could know how to use them and how they work but that doesn't really matter if you don't put them correctly where they need to go. Another takeaway is that small small mistakes can greatly affect how things work and it is always important to pay attention to what functions you are using and before typing any code you should always think to yourself and ask if this makes sense or what happens if I use this or that etc.
